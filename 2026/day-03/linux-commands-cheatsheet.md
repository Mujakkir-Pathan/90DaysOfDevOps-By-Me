# Linux Commands Cheat Sheet

## 1. Process Management

| Command | Usage |
|---|---|
| `ps aux` | Show all running processes |
| `top` | Monitor live system processes |
| `htop` | Interactive process viewer |
| `kill PID` | Stop a process using its PID |
| `jobs` | Show background jobs |
| `bg` | Run a stopped job in the background |
| `uptime` | Show system uptime and load |

---

## 2. File System Commands

| Command | Usage |
|---|---|
| `pwd` | Show current directory |
| `ls -la` | List all files with details |
| `cd directory` | Change directory |
| `mkdir folder` | Create a new directory |
| `rm -rf folder` | Delete a folder forcefully |
| `cp file1 file2` | Copy files |
| `mv file1 file2` | Move or rename files |
| `touch file.txt` | Create an empty file |
| `cat file.txt` | View file content |
| `vim file.txt` | Edit a file using the Vim editor |
| `df -h` | Show disk space usage |

---

## 3. Networking Troubleshooting

| Command | Usage |
|---|---|
| `ping google.com` | Check internet connectivity |
| `ip addr` | Show IP addresses |
| `curl https://example.com` | Test website response |
| `dig google.com` | Check DNS records |
| `traceroute google.com` | Trace the network path |
| `wget URL` | Download files from the internet |

---

## 4. Logs & System Information

| Command | Usage |
|---|---|
| `uname -a` | Show system information |
| `free -h` | Display memory usage |
| `history` | Show command history |
| `tail -f logfile.log` | Monitor a log file live |
| `whoami` | Show the current user |
| `hostname` | Display the system hostname |
